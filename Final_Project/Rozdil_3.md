# \## 3.2 Права доступу та безпека

# 

# <ul>

# <li><strong>Багаторівнева автентифікація:</strong> забезпечує безпечний вхід до системи.</li>

# <li><strong>Журнал активності користувачів:</strong> використовується для контролю та аудиту.</li>

# <li><strong>Шифрування даних:</strong> застосовується на рівні бази даних та при передачі даних.</li>

# <li><strong>Розмежування ролей:</strong> визначає доступність функцій і видимість даних для кожної групи користувачів.</li>

# </ul>
